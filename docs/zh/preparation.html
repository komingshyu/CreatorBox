环境变量 ​

Windows、Linux、MacOS ​

以下是需要配置的环境变量和相应的 API 密钥。请根据你的需求替换 <OPENAI_API_KEY>、<GOOGLE_API_KEY>、<DASHSCOPE_API_KEY>、<HUGGINGFACEHUB_API_TOKEN> 和 <NGROK_AUTH_TOKEN> 为实际的密钥。

所有信息均存储在本地电脑的环境变量中，程序不会记录保存 apikey 信息

1.右键点击 此电脑 并选择 属性

2.点击 高级系统设置，然后选择 环境变量

3.在 系统变量 部分，点击 新建 并分别添加以下变量：

  • OPENAI_API_KEY
  • GOOGLE_API_KEY
  • DASHSCOPE_API_KEY
  • HUGGINGFACEHUB_API_TOKEN
  • NGROK_AUTH_TOKEN

4.进入命令提示符或PowerShell，运行以下命令验证：

bash
echo $env:GOOGLE_API_KEY

提示

Colab 使用，见 《远程部署》 章节

多媒体处理 FFmpeg ​

FFmpeg 是一个功能强大、跨平台的开源命令行工具和开发库，用于处理音视频文件。它可以用来 录制、转换、剪辑、流式传输音视频，支持几乎所有常见的音视频格式

1. 下载安装 ​

前往官网根据操作系统下载 FFmpeg 安装包 https://ffmpeg.org

这里以 Windows 为例

  • 解压 ffmpeg-essentials_build.7z 压缩包至 D:\Program Files\FFmpeg 文件夹
  • 配置环境变量在 Path 中追加 D:\Program Files\FFmpeg\bin 注销重新登录

2. 验证安装 ​

打开 PowerShell，输入如下命令

bash
ffmpeg -version

推荐下载 ​

虚拟环境 Miniconda ​

Miniconda 是一个可以快速创建干净、隔离的 Python 虚拟环境，适合数据分析、AI 项目、自定义脚本等场景

1. 下载安装 ​

前往官网根据操作系统下载 Miniconda 安装包 https://www.anaconda.com/download/success

这里以 Windows 为例

  • 双击 .exe 安装包
  • 勾选 “Add Miniconda to PATH”（可选但推荐）
  • 完成后在开始菜单搜索 "Anaconda Prompt"

2. 验证安装 ​

bash
# 列出所有虚拟环境
conda env list

正常安装返回结果

log
PS C:\Users\home> conda env list
# conda environments:
#
base                   D:\Projects\miniconda3

3. 常见命令 ​

命令说明
conda create --name creatorbox python=3.11.11创建名为 creatorbox 的环境，指定 Py 版本 3.11.11
conda env list列出所有已创建的虚拟环境
conda activate creatorbox激活进入名为 creatorbox 的环境
conda deactivate退出当前激活的环境
conda remove -n creatorbox --all删除名为 creatorbox 的整个环境

注意

推荐使用 Miniconda 而非 Anaconda，体积更小、启动更快、依赖更纯粹

版本控制 Git ​

Git 是一个分布式版本控制系统，可以帮助开发人员跟踪代码更改、与他人协作以及高效管理软件项目

1. 下载安装 ​

前往官网根据操作系统下载 Git 安装包 https://git-scm.com

这里以 Windows 为例

  • 双击 .exe 安装包
  • 安装时推荐保留默认选项

2. 验证安装 ​

桌面右键选择 Git Bash Here，输入如下命令

bash
git --version

正常安装返回结果

log
git version 2.47.0.windows.2

3. 常见命令 ​

命令说明
git clone <仓库地址>克隆下载指定仓库地址文件
git pull拉取最新内容

注意

  • 1、git clone 克隆最好不要有空格、中文，避免一些奇葩问题

  • 2、git pull 进入克隆下载的 CreatorBox文件夹，运行此命令，可以直接更新代码
