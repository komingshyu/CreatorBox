配置选项 ​

模型 ​

每个提供商都有对应的模型默认值，用户也可以自定义

提供商默认值可选项
忽略--
OpenAigpt-3.5-turbogpt-3.5-turbo、gpt-4-turbo、o4-mini、o3-mini more
Geminigemini-2.0-flashgemini-2.5-pro、gemini-2.0-flash、gemini-1.5-pro、gemini-1.5-flash more
DashScopeqwen-turboqwen-max、qwq-plus more

忽略 选项有什么用？

  • 通过 调试 我们知道了返回的翻译结果，但总有个别翻译需要手动调整，此时我们可以单独修改 text_trans值并保存

  • 在后续执行中，想要使用我们手动调整的翻译，此时可选择 忽略选项

随机性 ​

控制文本生成的随机性，低温度更准确，高温度更具创意

多样性 ​

控制生成时探索的深度，较大值通常生成更准确的文本

批处理 ​

控制返回内容的长度，对于超长文本分批次处理

缓存 ​

开启后可节约 token 使用，优先读取缓存
